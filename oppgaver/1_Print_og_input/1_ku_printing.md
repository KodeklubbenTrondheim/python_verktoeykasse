## Beskrivelse

_Vanskelighetsgrad: Enkel_

La oss printe en ku som ser slik ut:

``` 
^__^
(oo)\_______
(__)\       )
    ||----W |
    ||     ||
``` 

For å printe den første linjen kan man skrive:

```
print("^__^")
```

## Oppgave

Print ut hele kua.
